Code Hound

An automated code review utility to support the development of efficient software more quickly
Watch Our Demo


flash_on

Speeds up development

Code reviews can be a long and arduous process. It can take a while before a fellow developer takes time out of their schedule to ensure that your code is sufficient and can be merged with the code base. Code Hound helps you get your code merged even faster and more accurately by making you aware of various style issues and code smells.

cloud

Nothing to Install

The Code Hound application is a SaaS offering that is hosted centrally on our servers. This means that you will always have the latest version of the product. Any bug that you're experiencing will be promptly considered as all other users are experiencing it as well.

settings

Easily Configurable

Configuration for all of your GitHub repositories can be done directly from the website. The application will pull down your organization's repositories and you can select the code smells that you want to scan for. The application allows you to run both a full repository review and configure automated reviews to occur on detection of pull requests.

Team Bio

We are a team of college students who paired up to create Code Hound as our senior project. We will most likely cease development on this project after we graduate. However, based on the GitHub community reaction we may reconsider that notion. If you feel that you may have useful contributions to this project, feel free to click the link to our projects on GitHub at the very bottom of the page. We feel that Code Hound could be a very viable solution for many software companies looking to expedite their software development lifecycle.
